# Notification

Handles email notifications triggered by domain events.

## Language

**Benachrichtigung** (Notification):
An email sent to a User when a significant event occurs. In German, the canonical term is _Benachrichtigung_.
_Avoid_: Alert, Message, Update

**Review-Benachrichtigung** (Review Notification):
An email sent when a new Bewertung is received for the User's Standort. Contains the Bewertung text, star rating, and a link to the Inbox to respond.
_Avoid_: New Review Alert, Incoming Review Email

**Präferenz** (Preference):
User-configurable settings controlling which Benachrichtigungen are sent and at what frequency (immediate, daily digest, off).
_Avoid_: Settings, Config, Opt-in
